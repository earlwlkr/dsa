# NiceTrees

Source screenshot: [nice-trees-1.png](nice-trees-1.png)

Find the minimum number of trees that need to be shortened so that the heights of all of them increase and decrease alternately.

## Description

Jimmy owns a garden in which he has planted `N` trees in a row. After a few years, the trees have grown up and now have different heights.

Jimmy finds his trees aesthetically pleasing if they alternately increase and decrease in height, for example: shorter, taller, shorter, taller, and so on.

The source screenshot contains visual examples of aesthetically pleasing and not aesthetically pleasing tree sequences.
